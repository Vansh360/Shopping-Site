# 🛒 Flipkart Clone — HTML, CSS & JavaScript

A simple, responsive Flipkart-inspired e-commerce front-end built with **pure HTML, CSS & JavaScript**.

## 🚀 Features

- 🏠 **5 Pages** — Home, Electronics, Grocery, About, Contact (no page reload)
- 🔝 **Sticky Navbar** — Search bar, nav links, login button, dark mode toggle
- 🔑 **Login Modal** — Username, Email, Password with backdrop blur
- 🛍 **Product Cards** — Images, badges, prices, discounts & Add to Cart
- 🌙 **Dark Mode** — CSS variables + saved to localStorage
- 🔔 **Toast Notifications** — Add to Cart, Login, Search feedback
- 📊 **Pricing Tables** — Electronics & Grocery price lists
- ☰ **Hamburger Menu** — Mobile-friendly slide-out drawer
- 📍 **Contact Page** — Google Maps, YouTube video & audio player
- 📱 **Fully Responsive** — CSS Grid, Flexbox & media queries

---

## ⚙️ How to Run

```bash
git clone https://github.com/your-username/flipkart-clone.git
cd flipkart-clone
open index.html
```

Or just **double-click** `index.html` — no setup needed!
