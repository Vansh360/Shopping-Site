# 🛒 Flipkart Clone – HTML, CSS & JavaScript

A simple and responsive Flipkart-style e-commerce front-end built using **HTML**, **CSS**, and **JavaScript**.  
It includes category pages, login modal, dark mode support, product listings, and a dynamic navigation system.


## 🚀 Features

### ✔️ Multi-Page Navigation (Single Page UI)
- Home
- Electronics
- Grocery
- About
- Contact  
(Sections displayed dynamically using JavaScript.)

### ✔️ Responsive Navigation Bar
- Search bar  
- Navigation buttons  
- Flexbox layout for responsiveness

### ✔️ Login Popup Form
- Username, Password & Email fields  
- Open/Close actions using JavaScript  

### ✔️ Categories Section (Home Page)
- Electronics  
- Fashion  
- Home Appliances  
- Footwear  
- Bags  
- Mobiles  

### ✔️ Product Sections
#### 📱 Electronics
- Product cards with hover effects  
- Images, titles, descriptions  
- Electronics pricing table  

#### 🥗 Grocery
- Grocery product cards  
- Grocery table with item & price  

### ✔️ Dark Mode Support
Using CSS `:root` variables for:
- background  
- text  
- cards  
- navbar  

### ✔️ Fully Responsive
- Mobile-friendly  
- Uses flexbox & media queries
